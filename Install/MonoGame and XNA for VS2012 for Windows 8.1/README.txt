Before install extension:
1. Install Games for Windows Live
2. Install MonoGame
   a) Run C:\Program Files (x86)\MonoGame\v3.0\oalinst.exe
3. Install Windows Phone 7.1 SDK (Necessary to load content projects for any XNA game)
4. Install DirectX
5. Install XNA Framework 4.0 Redistribution
6. Install XNA Game Studio Platform Tools
7. Install XNA Game Studio 4.0 2012 Extension

If you are getting an error "Error \PSM\IgnoreMe.dll could not be found"
Open the solution properties, go to Configuration Properties and check the build option for the content project
